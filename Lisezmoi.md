# Projet Gestion de Magasin

Ce projet est une application web de gestion de magasin développée dans un cadre pédagogique.
Il permet de gérer efficacement les produits, le stock et les opérations courantes d’un magasin.

---

##  Objectif du projet

L’objectif de ce projet est de mettre en pratique les concepts suivants :
- Développement web côté serveur avec PHP
- Manipulation d’une base de données relationnelle MySQL
- Gestion du stock et des données métiers
- Organisation d’un projet web complet

---

##  Technologies utilisées

- PHP
- MySQL
- HTML
- CSS
- phpMyAdmin
- Serveur local (XAMPP / WAMP)

---

##  Fonctionnalités principales

- Authentification (administrateur)
- Gestion des produits
- Gestion du stock
- Ajout, modification et suppression des données
- Interface simple et intuitive

##  Base de données

La base de données est fournie sous forme d’un fichier SQL exporté depuis phpMyAdmin.

 Emplacement : database/gestion_magasin.sql

##  Installation du projet

1. Copier le dossier du projet dans le répertoire `htdocs`
2. Lancer le serveur local (XAMPP ou WAMP)
3. Ouvrir phpMyAdmin
4. Créer une base de données nommée `gestion_magasin`
5. Importer le fichier `database/gestion_magasin.sql`
6. Accéder au projet via : http://localhost/gestion_magasin
